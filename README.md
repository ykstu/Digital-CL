# Digital-CL
> [!NOTE]
> Digital CL(Digital Craft Launcher)是一款由[Yangko Studio](https://github.com/ykstu)开发的，基于electron的Minecraft启动器，旨在为Windows,macOS和Linux平台用户提供高效启动体验。

## 构建
```
git clone https://github.com/ykstu/Digital-CL.git
cd Digital-CL
npm run test
```

## 进程
- [ ] 基本初始化
- [ ] 启动功能
- [ ] 下载功能
